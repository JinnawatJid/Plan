# ระบบการทำงาน (SOP - Standard Operating Procedure)

คู่มือปฏิบัติการเพื่อความสม่ำเสมอและคุณภาพงาน (System Checklist)

---

## 🏗️ A. หมวดเตรียมตัว (Setup Phase)
*ทำครั้งเดียวจบ หรือทำเมื่อมีการเปลี่ยนแปลงใหญ่*

- [ ] **1. Profile Optimization:**
    - [ ] รูปโปรไฟล์ชัดเจน เห็นหน้า (สร้างความน่าเชื่อถือ)
    - [ ] Bio เขียนกระชับ บอกว่าเราเป็นใคร และคนดูจะได้อะไร (เช่น "เพื่อนคู่คิดเรื่องเงิน ง่ายๆ สไตล์มนุษย์เงินเดือน")
    - [ ] ใส่ลิงก์ (Linktree/Gumroad) ใน Bio

- [ ] **2. Product Setup (Gumroad):**
    - [ ] สร้างสินค้า Digital Product (Budget Excel / Bingo Sheet)
    - [ ] ตั้งราคา "Pay what you want" (0+)
    - [ ] เขียนคำโปรยสินค้าให้คนอยากกดโหลด
    - [ ] ทดลองโหลดเอง 1 ครั้ง เพื่อเช็คความเรียบร้อย

---

## 🎬 B. หมวดผลิต Content (Production Loop)
*ทำทุกสัปดาห์ (Cycle: คิด -> เขียน -> ถ่าย -> ตัด -> โพสต์)*

### 1. Planning & Scripting (วันจันทร์-อังคาร)
- [ ] **เลือกหัวข้อ:** ดูจากแผนรายเดือน หรือกระแสช่วงนั้น
- [ ] **เขียนสคริปต์:** ใช้โครงสร้าง Hook -> Value/Story -> Call to Action (CTA)
- [ ] **ตรวจสอบ:** อ่านออกเสียงดูว่าติดขัดไหม ภาษาเป็นธรรมชาติหรือเปล่า

### 2. Filming (วันพุธ-พฤหัสบดี)
- [ ] **สถานที่:** แสงสว่างพอ หน้าไม่มืด ฉากหลังไม่รก
- [ ] **อุปกรณ์:** เช็ดเลนส์กล้องมือถือให้สะอาด (สำคัญมาก!)
- [ ] **การถ่าย:**
    - [ ] ถ่ายเผื่อหัว-ท้ายคลิปไว้นิดหน่อย
    - [ ] พูดช้ากว่าปกตินิดนึง ชัดถ้อยชัดคำ
    - [ ] ยิ้มแย้ม เป็นกันเอง

### 3. Editing (วันศุกร์-เสาร์)
- [ ] **ตัดต่อ:** ตัดช่วงเงียบ (Dead air) ออก ให้กระชับ
- [ ] **Captions:** ใส่ซับไตเติ้ลให้อ่านง่าย (คนส่วนใหญ่ดูคลิปแบบปิดเสียง)
- [ ] **Visuals:** ใส่ภาพประกอบ/B-roll เมื่อพูดถึงสิ่งที่ต้องเห็นภาพ (เช่น ตาราง Excel)
- [ ] **Music:** ใส่เพลงประกอบเบาๆ (ระวังลิขสิทธิ์ เลือกจากคลังของ App)

### 4. Publishing (วันอาทิตย์ หรือตามตาราง)
- [ ] **Cover:** เลือกปกคลิปให้น่ากด (เห็นหน้าชัด + ข้อความพาดหัวโดนๆ)
- [ ] **Caption:**
    - [ ] พาดหัวให้น่าสนใจ (เหมือน Hook ในคลิป)
    - [ ] สรุปเนื้อหาสั้นๆ
    - [ ] ใส่ CTA (เช่น "โหลดไฟล์ฟรีที่ลิงก์หน้าโปรไฟล์")
    - [ ] ใส่ Hashtag ที่เกี่ยวข้อง (3-5 อัน)

---

## 📊 C. หมวดดูแลและวัดผล (Maintenance & Review)
*ทำทุกสัปดาห์และทุกเดือน*

### 1. Daily/Weekly Tasks (งานรายวัน/สัปดาห์)
- [ ] **Reply:** ตอบคอมเมนต์ทุกอัน (ในช่วงแรก) เพื่อสร้างฐานแฟน
- [ ] **Engagement:** ไปคอมเมนต์/ไลก์ช่องอื่นในหมวดเดียวกันบ้าง
- [ ] **Tracking:** จดบันทึกผลลัพธ์ลงใน `TRACKING.md` (ทุกวันอาทิตย์)

### 2. Monthly Review (รีวิวรายเดือน)
- [ ] **Analytics:** ดูว่าคลิปไหนยอดวิวสูงสุด? เพราะอะไร?
- [ ] **Conversion:** ยอดดาวน์โหลดสินค้าเป็นยังไง?
- [ ] **Planning:** วางแผน Content เดือนถัดไป โดยอิงจากสิ่งที่เวิร์คในเดือนนี้
- [ ] **Product Improvement:** มีคอมเมนต์ติชมสินค้าไหม? ต้องปรับปรุงเวอร์ชั่น 2 หรือไม่?

*Tip: "วินัยสำคัญกว่าแรงบันดาลใจ" ทำตาม Checklist นี้ไปเรื่อยๆ ผลลัพธ์จะตามมาเองครับ!*
