# คู่มือการผลิตวิดีโอ (Video Production Guide)
*Zero Financial Budget Approach*

เอกสารนี้รวบรวมเครื่องมือฟรีและเทคนิคพื้นฐานในการถ่ายทำและตัดต่อวิดีโอ เพื่อให้ผู้บริหารและผู้ติดตามเข้าใจ Case Study ของคุณได้ง่ายที่สุด

## 1. เครื่องมืออัดหน้าจอคอมพิวเตอร์ (Screen Recording for B-Roll)
เพื่อความคมชัด ไม่ควรใช้กล้องมือถือถ่ายหน้าจอคอมพิวเตอร์ แต่ให้ใช้โปรแกรมที่ติดมากับเครื่อง (Native Tools)

*   **สำหรับผู้ใช้ Windows:**
    *   **เครื่องมือ:** Xbox Game Bar (มีในตัว) หรือ Snipping Tool (เวอร์ชันใหม่)
    *   **คีย์ลัด:** กด `Windows + G` เพื่อเปิด Game Bar แล้วคลิกปุ่มบันทึกวิดีโอ (หรือกด `Windows + Alt + R` เพื่อเริ่ม/หยุดทันที)
*   **สำหรับผู้ใช้ Mac:**
    *   **เครื่องมือ:** Screenshot Utility (มีในตัว)
    *   **คีย์ลัด:** กด `Command + Shift + 5` จะมีแถบเมนูปรากฏขึ้น เลือก "Record Entire Screen" หรือ "Record Selected Portion" แล้วกด Record

**💡 เทคนิคการอัด B-Roll ที่ดี:**
*   ซูมหน้าเว็บเพจ (กด Ctrl/Cmd + "+") ให้ตัวหนังสือใหญ่ขึ้นกว่าปกติเล็กน้อย เพื่อให้คนที่ดูผ่านจอมือถือเห็นชัดเจน
*   ขยับเมาส์อย่างช้าๆ มีเป้าหมาย ไม่ส่ายไปมา
*   ถ่ายเผื่อเวลาหัวและท้ายไว้เล็กน้อย เพื่อให้ง่ายต่อการตัดต่อ

## 2. เครื่องมือตัดต่อวิดีโอ (Video Editing)
แนะนำให้ใช้แอปพลิเคชันบนมือถือที่ฟรีและใช้งานง่ายที่สุดในปัจจุบัน

*   **แอปพลิเคชัน:** **CapCut** (รองรับทั้ง iOS และ Android)
*   *ถ้าชอบตัดในคอม:* สามารถโหลด CapCut for Desktop มาใช้ได้ฟรีเช่นกัน

## 3. เทคนิคการแทรก B-Roll (ภาพหน้าจอ) ในแอป CapCut

เมื่อคุณนำคลิปที่คุณยืนพูด (A-Roll) เข้ามาในแอปแล้ว คุณมี 2 ทางเลือกในการใส่ B-Roll:

### แบบที่ 1: ตัดสลับ (Cutaway) - *แนะนำสำหรับคลิปนี้*
วิธีนี้คนดูจะเห็นภาพหน้าจอคอมเต็มๆ ทำให้เห็น Prompt ชัดเจนที่สุด
1. นำเพลย์เฮด (เส้นสีขาว) ไปตรงจุดที่คุณพูดว่า *"วิธีทำก็แค่ลากไฟล์..."*
2. กดปุ่ม `Split (แบ่งฉาก)` ที่คลิปพูดของคุณ
3. เลื่อนเส้นไปตรงจุดที่คุณพูดจบประโยค แล้วกด `Split` อีกครั้ง
4. กดเครื่องหมาย `+` เพื่อเพิ่มคลิปวิดีโอหน้าจอ (B-Roll) แทรกตรงกลาง
5. ลบคลิปช่วงที่คุณพูด (ที่เพิ่งตัดแยกไว้) ออกไป

### แบบที่ 2: ภาพซ้อนภาพ (Picture-in-Picture / Overlay)
วิธีนี้คนดูจะยังคงเห็นหน้าคุณพูดไปด้วย ขณะที่มีจอคอมลอยอยู่ข้างๆ
1. นำเส้นสีขาวไปตรงจุดที่ต้องการให้ภาพแทรกขึ้นมา
2. ดูที่เมนูด้านล่าง กดคำว่า **`Overlay (ภาพซ้อน)`** > `Add Overlay`
3. เลือกคลิปอัดหน้าจอ (B-Roll) เข้ามา
4. ใช้นิ้วจีบย่อขยาย หรือลากคลิป B-Roll ให้ไปอยู่ตำแหน่งที่ไม่บังหน้าคุณ

## 4. การใส่ข้อความ (Text & Captions)
*   **สำหรับข้อความเน้นย้ำ (เช่น Prompt):** กดเมนู `Text` > `Add Text` แล้วพิมพ์ประโยคลงไป แนะนำให้เลือกฟอนต์หนาๆ อ่านง่าย เลี่ยงสีฉูดฉาดที่กวนสายตา
*   **คำบรรยายอัตโนมัติ (Auto Captions):** ใน CapCut มีฟีเจอร์ "Auto Captions" ที่จะฟังเสียงคุณแล้วสร้างซับไตเติ้ลให้ฟรี (รองรับภาษาไทย) แนะนำให้เปิดใช้เสมอ เพราะหลายคนดูคลิปแบบปิดเสียง
