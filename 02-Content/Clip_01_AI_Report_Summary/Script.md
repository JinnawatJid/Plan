# Shooting Script 01: แจก Prompt สรุปรีพอร์ตด้วย AI

**ความยาวเป้าหมาย:** 45 - 60 วินาที
**สไตล์:** พูดเร็ว กระชับ มั่นใจ เข้าเรื่องทันที เน้นให้ประโยชน์ (Value-First)
**โครงสร้าง:** Value Hook -> Situation -> Action -> Result & CTA

---

### [วินาทีที่ 0-5] - VALUE HOOK (แจกของ/ดึงความสนใจ)
*(แอคชั่น: มองกล้องด้วยความมั่นใจ หรือโชว์หน้าจอที่มี Prompt)*
🗣 **บทพูด:** "แจก Prompt สรุปเอกสาร และเขียน Action Plan ว่าต้องทำอะไรต่อบ้าง"

### [วินาทีที่ 6-15] - SITUATION (สร้างความรู้สึกร่วม)
*(แอคชั่น: ถือปึกกระดาษ หรือ ยิ้มแบบเข้าใจปัญหา)*
🗣 **บทพูด:** "เพื่อนๆ เคยเป็นกันมั้ยครับ... ต้องอ่านรีพอร์ตยาวเป็นสิบๆ หน้า แต่เรามีเวลาอ่านแค่ 2 นาที... ผมมีทริคง่ายๆ มาแชร์ครับ"

### [วินาทีที่ 16-35] - ACTION (สอนวิธีแก้ด้วยเทคโนโลยี)
*(แอคชั่น: อินเสิร์ทภาพหน้าจอ (B-roll) กำลังใช้งาน Claude ลากไฟล์ลงไป และพิมพ์คำสั่ง)*
🗣 **บทพูด:** "เราจะใช้ AI อย่าง Claude มาเป็นผู้ช่วยครับ วิธีทำก็แค่ลากไฟล์เอกสารลงไป แล้วพิมพ์ Prompt ตามนี้เลยครับ...

> *'ช่วยสรุปปัญหาหลัก 3 ข้อจากเอกสารนี้ และเขียน Action Plan ว่าใครต้องทำอะไรต่อ'*

*(ขึ้น Text ตัวหนังสือบนจอเป็น Prompt ให้คนดูแคป)*
รอแค่ 10 วินาที Claude ก็จะย่อยข้อมูลออกมาเป็นข้อๆ ให้เราเข้าใจง่ายขึ้นครับ"

### [วินาทีที่ 36-45] - RESULT & CTA (ผลลัพธ์ และ ปิดคลิป)
*(แอคชั่น: กลับมามองกล้อง ยิ้มแย้ม)*
🗣 **บทพูด:** "จากที่ต้องนั่งอ่านเองเป็นชั่วโมง เหลือเวลาตัดสินใจแค่ไม่กี่นาที ลองเอาไปปรับใช้ดูนะครับ
ถ้าชอบทริคการทำงานแบบนี้ ฝากกดติดตามไว้ด้วยนะครับ แล้วเจอกันคลิปหน้าครับ!"
