# Finance Side Hustle: From Zero to Hero

ยินดีต้อนรับสู่ Project ปั้นรายได้เสริมสายการเงิน!
Repository นี้ใช้สำหรับเก็บรวบรวมแผนงาน, ไอเดีย Content, และ Digital Product ต่างๆ เพื่อสร้างรายได้เสริมจากทักษะและความสนใจด้านการเงิน

## 🎯 เป้าหมายหลัก (The Goal)
สร้างรายได้เสริมจาก 0 -> 1,000 บาท/เดือน ภายใน 6 เดือน
โดยเน้นทำควบคู่กับงานประจำ (ใช้เวลาวันละ 1-2 ชั่วโมง)

## 📂 โครงสร้าง Project
- **01-Planning/**: แผนงานระยะยาวและรายเดือน รวมถึงไอเดียที่รอการพัฒนา
- **02-Content/**: สคริปต์วิดีโอ, แคปชั่น, และข้อมูลสำหรับทำ Content (ทรัพย์สินทางปัญญา)
- **03-Products/**: ไฟล์สินค้า Digital Product (เช่น Excel Template) และเครื่องมือต่างๆ

## 🚀 สถานะปัจจุบัน
**Phase 1: Validation (Month 1-2)**
- [ ] สร้าง Content เพื่อดึงดูดกลุ่มเป้าหมาย
- [ ] พัฒนาสินค้าชิ้นแรก (Simple Budget Planner)
- [ ] ทำยอดขายแรกให้ได้ (The First Dollar)
